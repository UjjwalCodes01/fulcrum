- **Grand Prize:** $5,000 + featured blog post on auth0.com/blog

## Core Requirement
**MUST USE:** Token Vault from Auth0 for AI Agents

## Theme & Vision
Build agentic AI applications where Auth0 handles:
- OAuth flows
- Token management
- Consent delegation
- Async authentication
- Step-up authentication

Focus: Let AI agents authenticate and interact with APIs/services securely, just like humans do.

## Judging Criteria (Equally Weighted)

### 1. Security Model
- Does agent operate within explicit permission boundaries?
- Are credentials protected and access properly scoped?
- High-stakes actions identified and protected?
- Step-up authentication where needed?

### 2. User Control
- Can users understand agent permissions?
- Is consent clearly communicated?
- Are scopes and access boundaries well-defined?

### 3. Technical Execution
- Quality implementation of Token Vault?
- Production-aware/production-ready?
- Demonstrates proper Auth0 patterns?

### 4. Design
- Well thought out UX/UI?
- Balanced frontend and backend?

### 5. Potential Impact
- Impact on AI developer community?
- Impact beyond target community?

### 6. Insight Value
- Surface useful patterns?
- Identify pain points or gaps?
- Inform evolution of agent authorization?

2. **User transparency** - clear consent and permission communication  
3. **Production quality** - not just a proof-of-concept
4. **Thoughtful design** - good UX/UI balance
5. **Real impact** - solves actual problems for AI developers
6. **Innovation** - new patterns or insights for agent authorization
